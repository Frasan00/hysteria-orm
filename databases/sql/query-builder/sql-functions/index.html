SQL Functions

Hysteria ORM provides shorthand methods for common SQL functions. These methods are fully type-safe and return typed results.

Aggregate Functions​

MethodSQL FunctionReturn TypeDescription
selectCountCOUNT(col)numberCount rows or non-null values
selectCountDistinctCOUNT(DISTINCT col)numberCount unique values
selectSumSUM(col)numberSum numeric values
selectAvgAVG(col)numberCalculate average
selectMinMIN(col)numberFind minimum value
selectMaxMAX(col)numberFind maximum value

Basic Usage​

// COUNT all rows
const result = await User.query()
  .selectCount('*', 'totalUsers')
  .one();

console.log(result?.totalUsers); // e.g., 42

// SUM
const result = await Order.query()
  .selectSum('amount', 'totalRevenue')
  .one();

// AVG
const result = await Product.query()
  .selectAvg('price', 'averagePrice')
  .one();

// MIN / MAX
const result = await User.query()
  .selectMin('age', 'youngestAge')
  .selectMax('age', 'oldestAge')
  .one();

// COUNT DISTINCT
const result = await User.query()
  .selectCountDistinct('status', 'uniqueStatuses')
  .one();
// result.uniqueStatuses = 3

Combining Aggregates​

const stats = await User.query()
  .selectCount('*', 'totalUsers')
  .selectMin('age', 'minAge')
  .selectMax('age', 'maxAge')
  .selectAvg('age', 'avgAge')
  .one();

// { totalUsers: 100, minAge: 18, maxAge: 85, avgAge: 35.5 }

With GROUP BY​

const salesByCategory = await Product.query()
  .select('category')
  .selectSum('price', 'totalSales')
  .selectCount('*', 'productCount')
  .groupBy('category')
  .orderBy('totalSales', 'desc')
  .many();

Alternative: getCount, getSum, etc.​

For simple cases where you just need a single aggregate value:

// Using getCount (returns just the number)
const count = await User.query()
  .where('status', 'active')
  .getCount();
// Returns: 42

// Equivalent using selectCount (returns model with property)
const result = await User.query()
  .where('status', 'active')
  .selectCount('*', 'count')
  .one();
// Returns: { count: 42 }

String Functions​

MethodSQL FunctionReturn TypeDescription
selectUpperUPPER(col)stringConvert to uppercase
selectLowerLOWER(col)stringConvert to lowercase
selectLengthLENGTH(col)numberGet string length
selectTrimTRIM(col)stringRemove leading/trailing whitespace

Usage​

// Convert to uppercase
const result = await User.query()
  .select('name')
  .selectUpper('name', 'upperName')
  .one();
// result.upperName = "JOHN DOE"

// Convert to lowercase
const result = await User.query()
  .selectLower('email', 'lowerEmail')
  .one();
// result.lowerEmail = "john@example.com"

// Get string length
const result = await User.query()
  .select('name')
  .selectLength('name', 'nameLength')
  .one();
// result.nameLength = 8

// Trim whitespace
const result = await User.query()
  .selectTrim('name', 'trimmedName')
  .one();
// result.trimmedName = "John" (removes leading/trailing spaces)

Numeric Functions​

MethodSQL FunctionReturn TypeDescription
selectAbsABS(col)numberAbsolute value
selectRoundROUND(col, n)numberRound to n decimal places
selectCeilCEIL(col)numberRound up to nearest integer
selectFloorFLOOR(col)numberRound down to nearest integer
selectSqrtSQRT(col)numberSquare root

Usage​

// Absolute value
const result = await Order.query()
  .select('balance')
  .selectAbs('balance', 'absoluteBalance')
  .one();
// If balance = -100, result.absoluteBalance = 100

// Round to decimal places
const result = await Order.query()
  .selectRound('price', 2, 'roundedPrice')
  .one();
// result.roundedPrice = 19.99

// Ceiling (round up)
const result = await Order.query()
  .selectCeil('price', 'ceilPrice')
  .one();
// If price = 19.1, result.ceilPrice = 20

// Floor (round down)
const result = await Order.query()
  .selectFloor('price', 'floorPrice')
  .one();
// If price = 19.9, result.floorPrice = 19

// Square root
const result = await Data.query()
  .selectSqrt('value', 'sqrtValue')
  .one();
// If value = 100, result.sqrtValue = 10

Null Handling​

MethodSQL FunctionReturn TypeDescription
selectCoalesceCOALESCE(col, default)anyReturn default if NULL

Usage​

// COALESCE - return default when NULL
const result = await User.query()
  .selectCoalesce('nickname', "'Unknown'", 'displayName')
  .one();
// If nickname is NULL, result.displayName = "Unknown"

// With numeric default
const result = await Order.query()
  .selectCoalesce('discount', 0, 'appliedDiscount')
  .one();
// If discount is NULL, result.appliedDiscount = 0

Chaining Multiple Functions​

All SQL function methods can be chained together:

const result = await User.query()
  .select('name')
  .selectUpper('name', 'upperName')
  .selectLength('name', 'nameLength')
  .selectCount('*', 'total')
  .selectCountDistinct('status', 'uniqueStatuses')
  .one();

// All values are typed correctly:
// result.name: string
// result.upperName: string
// result.nameLength: number
// result.total: number
// result.uniqueStatuses: number

Database Compatibility​

warning

Some functions have database-specific limitations:

FunctionLimitation
selectRound(col, n)PostgreSQL/CockroachDB: Requires NUMERIC type column, not REAL/FLOAT
selectCeilSQLite: Not supported (no native CEIL function)
selectFloorSQLite: Not supported (no native FLOOR function)
selectSqrtCockroachDB: Requires FLOAT type column, not INT
selectLengthMSSQL: Uses LEN() which does not count trailing spaces
selectCeilMSSQL: Automatically uses CEILING() instead of CEIL()

Raw Select (selectRaw)​

For complex SQL expressions not covered by the built-in methods, use selectRaw():

const result = await User.query()
  .selectRaw<{ total: number }>('COUNT(*) as total')
  .one();

console.log(result?.total);

Type Parameter​

Always provide a type parameter for TypeScript support:

// With type parameter - full type safety
const result = await User.query()
  .selectRaw<{ avgAge: number; maxAge: number }>(
    'AVG(age) as avgAge, MAX(age) as maxAge'
  )
  .one();

result?.avgAge; // number
result?.maxAge; // number

Common Use Cases​

// Mathematical expressions
const result = await Product.query()
  .selectRaw<{ discountedPrice: number }>(
    'price * (1 - discount_rate) as discountedPrice'
  )
  .many();

// String functions (PostgreSQL)
const result = await User.query()
  .selectRaw<{ fullName: string }>(
    "CONCAT(first_name, ' ', last_name) as fullName"
  )
  .many();

// Date functions
const result = await Order.query()
  .selectRaw<{ orderYear: number }>(
    'EXTRACT(YEAR FROM created_at) as orderYear'
  )
  .groupBy('orderYear')
  .many();

// CASE expression
const result = await User.query()
  .selectRaw<{ ageGroup: string }>(
    "CASE WHEN age < 18 THEN 'minor' WHEN age < 65 THEN 'adult' ELSE 'senior' END as ageGroup"
  )
  .many();

CAST Expressions​

CAST expressions are fully supported. The type after AS inside CAST() is correctly recognized as a SQL type, not an alias:

const result = await User.query()
  .selectRaw<{ ageString: string }>(
    'CAST(age AS VARCHAR) as ageString'
  )
  .one();

Combining with Regular Select​

const result = await User.query()
  .select('name', 'email')
  .selectRaw<{ accountAge: number }>(
    'DATEDIFF(NOW(), created_at) as accountAge'
  )
  .many();

result[0].name;       // From select()
result[0].email;      // From select()
result[0].accountAge; // From selectRaw()
